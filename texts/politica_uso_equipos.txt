La política de uso de equipos bajo ISO 27001 establece lineamientos claros para garantizar la seguridad y el correcto uso del hardware y software corporativo. Para los usuarios visitantes, se permite el uso de dispositivos personales bajo estrictos controles, donde solo se habilitan puertos específicos para tareas autorizadas y se prohíbe la instalación no autorizada de aplicaciones o almacenamiento de datos corporativos en dichas máquinas. Se requiere que los visitantes conecten sus equipos mediante una red separada y protegida para minimizar riesgos.

Para usuarios administrativos, se restringe el uso exclusivamente a dispositivos corporativos con acceso limitado. El uso de equipos está supervisado, y está prohibida la instalación de programas sin consentimiento explícito del área de TI para evitar vulnerabilidades. Los desarrolladores requieren equipos con configuraciones personalizadas y permisos para instalar software específico necesario para su labor, bajo monitoreo permanente y registros de actividad.

Los usuarios gerenciales y directivos pueden utilizar dispositivos corporativos o personales con políticas menos estrictas, siempre garantizando que tengan instalados sistemas antivirus, cortafuegos, y actualizaciones al día. Para usuarios móviles en Android e iOS, se establecen controles enfocados en el cifrado del dispositivo, uso de VPN para acceso remoto seguro, y la instalación solo de apps aprobadas para minimizar riesgos de malware.

Todos los equipos conectados a la red corporativa deberán seguir protocolos de actualización, respaldo, y tener instalado software antivirus y sistemas de detección de intrusiones para monitorear amenazas en tiempo real. La política también contempla la retirada segura de dispositivos para evitar filtraciones de información cuando son desechados o reasignados.
